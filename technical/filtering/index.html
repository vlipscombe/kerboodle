Web and Email Filtering

Welcome to your Kerboodle browser support page. This page answers the most common queries from our users about web and email filtering. You may also like to look at the Technical support page for system requirements, and other technical queries.

If you cannot resolve your query, please Contact us.

Frequently asked questions

Q: Why do I need to whitelist email addresses?

Every Teacher and Administrator activates their accounts through an activation email sent from us. If the teacher does not receive the activation email then they will not be able to access their account. The email address which sends out the activation email is no-reply@kerboodle.com; please add this email address to your school’s accepted list of emails. We realise that some email can be misinterpreted as spam so please ensure that the teacher checks these folders before contacting us.

Q: Which email addresses do I need to whitelist?

The email address which sends out the activation email is no-reply@kerboodle.com; please add this email address along with support@kerboodle.com to your school’s accepted list of emails. We realise that some email can be misinterpreted as spam so please ensure that the teacher checks these folders before contacting us.

Q: Why do I need to whitelist certain web URL?

We understand that schools need a high level of security when it comes to internet use within a school. Unfortunately some of these security features can sometimes have an effect on the loading of some Kerboodle pages and content. To avoid any potential issues like this you can add the Kerboodle domains and URLs to the school’s web filtering software to exclude them from the filtering process. We try to keep our list as up to date as possible, please email us at support@kerboodle.com and ask for the latest list, if the problems still persist after adding them then please contact your Web Filtering provider to ensure the settings that are applied are correct.

You can run your own tests with the developer tools within certain browsers to see which URLs are being blocked. If you want to find out more information about these developer tools then please click on the links provided below.

  • Chrome Developer tools
  • Firefox Developer tools
  • Internet Explorer Developer tools

Q: How do I get the latest list of domains to whitelist??

A: Please email support@kerboodle.com and asking for the latest whitelist for domains and emails addresses.
