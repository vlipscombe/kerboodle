Technical - Getting Started

This getting started guide has been created for the use of your school’s IT Department. We want all users to be able to use Kerboodle to its full extent, so we have designed this page to highlight some of the technical configuration that needs to be applied to your school networks.

System Requirements

Please ensure that the machines and devices that the teachers and students use fit in with System Requirements. If you are unsure of the details of your machine/device then you can try out our new System Requirement Checker which will run a series of tests to see if it is compatible.

Emails

Every Teacher and Administrator activates their accounts through an activation email sent from us. If the teacher does not receive the activation email then they will not be able to access their account. The email address which sends out the activation email is no-reply@kerboodle.com; please add this email address to your school’s accepted list of emails. We realise that some email can be misinterpreted as spam so please ensure that the teacher checks these folders before contacting us.

Another email address to add to the allowed list is support@kerboodle.com, which is our main support email address.

Web Filtering

We understand that schools need a high level of security when it comes to internet use within a school. Unfortunately some of these security features can sometimes have an effect on the loading of some Kerboodle pages and content. To avoid any potential issues like this you can add the Kerboodle domains and URLs to the school’s web filtering software to exclude them from the filtering process. We try to keep our Domain list as up to date as possible, if the problems still persist after adding them then please contact your Web Filtering provider to ensure the settings that are applied are correct.

You can run your own tests with the developer tools within certain browsers to see which URLs are being blocked. If you want to find out more information about these developer tools then please click on the links provided below.
