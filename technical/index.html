Login

Q: I am using the correct username and password but I still can’t login.

A: Please check your cookie settings as the online products requires cookies to be enabled to use the site. If you are unsure how to set the cookie setting on your school network then please consult your IT Department.

If you are using a personal computer then the guides below can help you to enable cookies.

Internet Explorer - Help

Mozilla Firefox - Help

Google Chrome - Help

Safari - Help

Ipad - To set your Ipad to allow cookies please click on Setting > Safari > Privacy > Accept cookies

If you would like more information about how we use cookies then please visit our Cookies Policy Page

A:

Web and email filtering setup

Q: Will I need to make any exceptions to my web filtering?

A:

Please ask your IT department to whitelist the *.oxfordsecondary.co.uk. If you encounter any further issues relating to web filtering then please contact us for a full list of domains.

Q: Will I need to whitelist any email addresses?

A:

Please make sure that the email address edtechsupport@oup.com has been added to your whitelist/safe senders list. This email address will send teachers and the administrator their login details. You will also receive emails from this address if you request any support for this product.
