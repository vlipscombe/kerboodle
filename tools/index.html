Tools

We are always trying to create useful tools to make the process of using Kerboodle easier. Please see some of our latest additions below -

  • The System Requirements Check will test your current system to check to see if you have all the required software and plug-ins to run Kerboodle
  • You can use the Student Slips Generator to create printable login slips for your students, all you will need is the CSV file used to upload your students.
